如何创建一个vue项目

发表于 2020-04-17   |   分类于 总结
    1. 安装vue-cli

      vue uninstall -g vue/cli \\卸载vue-cli\
      vue install -g @vue/cli \\安装全局vue-cli\
      vue --version \\查看vue版本

  • 2.(2.x版本):vue init webpack [项目名称](3.x版本)：vue create [项目名称]
  • 3.(2.x版本)npm run dev(3.x版本)npm run serve本地预览
  • vue uiGUI界面
  • 4.npm run build创建工程，可上传至服务器
  • 注意：在遇到Unexpected end of JSON input while parsing near使用下面的命令可能会解决
  • npm cache clean --force//强制清除缓存

  2. 重新build
